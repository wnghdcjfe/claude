## 🔴 Critical — 즉시 수정 필요

### C1. 타이틀 배너가 영상 전체(0~90s) 정중앙을 가린다
**증상**: "5월 14일 데이브리핑" 노란색 글자 + 검정 박스 배너가 영상 모든 프레임의 캔버스 정중앙에 박혀 있어, 호랑이 일러스트·차트·"사상 최고치"/"AI 계약"/"랠리"/"소비심리 추락" 등 핵심 비주얼 텍스트를 가린다.
샘플링한 18개 프레임 중 14개에서 일러스트 메인 영역(호랑이 얼굴·몸통)을 직접 가림.

**영향**: 스크립트가 설명하는 이미지(주요 시각 정보)가 보이지 않음 → 영상이 "음성 + 자막"만 남고 비주얼이 무력화됨.

**개선**:
- 데이브리핑 타이틀은 오프닝(0~1.5s)에서만 풀스크린 카드로 노출하고 그 이후에는 제거.
- 유지해야 한다면 상단 작은 헤더(MD2Short 로고와 결합) 또는 하단 코너 워터마크로 축소.
- Remotion `props.json`에서 title overlay의 `duration_ms`를 전체 길이가 아닌 인트로 구간만으로 한정.

### C2. 인트로 0~9초가 사실상 데드 에어
**증상**: frame_002~frame_003 구간은 회색 배경에 타이틀 박스만 있는 정지 카드. 음성도 늦게 시작.

**영향**: 숏폼은 첫 1~3초에 이탈률이 결정됨. 현재 후킹 요소 0.

**개선**:
- 첫 1초 안에 가장 자극적인 헤드라인 한 줄(예: "나스닥 사상 최고치 🚀")을 강한 모션으로 노출.
- 인트로 카드 자체를 1.5초 이하로 단축, 즉시 본문 비주얼로 전환.
- 인트로 사운드(swoosh) + 핵심 수치(예: `S&P 5,723 / +0.93%`) 큰 글씨 후킹.

### C3. 자막 청크 분할이 한국어 어절 경계를 무시
**증상**:
- frame_009: `"에스엔피 오백 종가가칠이삼영을 / 기록했고,"`
- frame_022: `"이는 정부의 AI 시장이 본격적인 매출 / 사이클에 진입했다는 신호로 해석됩니다."`
- frame_020: `"정부 AI 관련 소식 오라클이 미 국방부의 / AI 네트워크 프로젝트에 공식 / 합류했습니다."` (3줄 점유)
- 단어 중간 분리: `"사 월"`(사월), `"오 주 연속"`(5주 연속), `"더 자세한"` 등

**원인**: 글자 수 기준 청크 분할(`chunk_size=14`)이 한국어 어절·형태소 경계를 무시.

**개선**:
- 어절·문장부호 단위로 줄바꿈을 끊는 `wrap_korean_caption()` 추가.
- 한 화면 최대 2줄·줄당 14자 이내 유지하되, 어절 중간 절단 금지.
- `words.json`의 word-level 타임스탬프를 활용해 "한 호흡(pause) = 한 캡션 청크"로 묶기.

## 🟠 High — 시청 완료율 직격

### H1. 숫자·영문 표기 일관성 붕괴 (TTS 한글 음차 그대로 자막에 노출)
**증상**:
- `"에스엔피 오백 종가가칠이삼영을"` ← S&P 500 종가 6,723.40
- `"영돗구상퍼센트"` ← +0.93%
- `"일팔퍼센트"` ← -18%
- `"일구오년 이후"` ← 1995년 이후
- 한편 `"AI 계약"`, `"AI 시장"`, `"AI 네트워크"`는 영문 그대로 유지됨 → **표기 룰이 일관되지 않음**.

**영향**: 시청자가 즉시 "0.93%인지 0.93포인트인지" 인지 불가. 숏폼 데이터 영상의 가장 큰 무기인 "숫자"가 사라짐.

**개선**:
- 자막 렌더링은 원본 마크다운의 숫자/단위 토큰(`5,723.0`, `+0.93%`, `-18%`)을 그대로 표시.
- 음성용 텍스트(TTS input)와 자막용 텍스트(caption)를 **분리** — Script Engine에서 두 가지를 동시에 생성.
  - `spoken_text`: TTS 친화 (예: "에스앤피 오백 지수가 오천 칠백 이십삼 포인트로...")
  - `display_text`: 자막용 원본 표기 (예: "S&P 500 지수가 5,723.0pt로...")
- 표기 화이트리스트: 종목명·지수명·티커·숫자·단위는 자막에서 영문/숫자 원형 유지(`S&P 500`, `AI`, `1995년`, `+0.93%`).
- 핵심 수치는 화면 우상단에 크게 별도 표기(데이터 칩 — H3 참조).

### H2. 비주얼이 정적·반복적
**증상**:
- 호랑이가 신문 읽는 동일 일러스트가 frame_001 / frame_008 / frame_015에 반복.
- 전 프레임 zero motion (확대·이동·페이드 없음).
- 단일 컷이 5~10초씩 유지됨 → 숏폼 권장 컷 길이(1.5~3초)의 3배 이상.

**영향**: 숏폼 알고리즘이 선호하는 "변화율"이 낮아 노출 추천에 불리. 시청자 체감 이탈률↑.

**개선**:
- Remotion `spring()` 또는 `interpolate()`로 일러스트에 Ken Burns(서서히 줌·이동) 적용.
- 동일 일러스트 재사용 시 최소한 좌우반전·컬러 시프트로 다양화.
- 장면 전환에 200~300ms 페이드/슬라이드.
- 한 문장당 1컷 원칙 — `words.json` pause boundary 기준으로 자동 컷 분할.

### H3. 데이터 시각화 부재
**증상**: "S&P 5,723", "나스닥 +0.93%", "다우 -소폭", "로블록스 -18%" 등 핵심 수치가 음성으로만 전달되고 차트·바·증감 화살표가 없음.

**개선**:
- `visual_mode=emoticon` 외에 `visual_mode=metric_card` 추가 — 수치·티커·증감을 Remotion 컴포넌트로 렌더.
- 최소한 +/-를 색(빨강/초록)과 화살표로 강조.
- Script Engine 단계에서 마크다운의 숫자/티커 토큰을 추출 → 자동으로 `metric_card` 컴포넌트로 매핑.

### H4. 캔버스 활용률 저조 (위·아래 검은 띠)
**증상**: 1080×1920 캔버스에 회색 일러스트 박스는 약 60% 영역만 차지하고 상단 100px·하단 200px가 검정 빈 공간. "MD2Short" 로고가 검은 띠 위에 외롭게 떠있음.
일러스트가 없는 컷(frame_011, frame_016)에선 회색만 가득 차서 "로딩 실패" 같은 인상을 줌.

**개선**:
- 일러스트 배경을 풀블리드(1080×1920 전면)로 확장하거나, 검정 띠를 의도적 디자인(그라데이션·블러)으로 채움.
- "MD2Short" 워드마크는 안전 영역 안 우상단 코너로 축소(48px 정도).

### H5. 영상 길이 89.77초 — 숏폼 기준 길다
**증상**: TikTok·Reels·Shorts 최적 길이는 30~60초. 현재 90초에 가까움.

**원인**:
- 인트로 데드 에어 7~9초 (C2 참조).
- 무음 구간 누적 약 18~20초 (M1 참조).
- 단일 컷 5~10초 유지로 늘어짐 (H2 참조).

**개선**:
- 목표 길이 30~45초.
- C2 + M1 + H2를 함께 적용하면 자동 압축 가능.
- 그래도 길어지면 "긴 버전(채널)"과 "30초 hook 버전(숏폼)"을 동일 마크다운에서 분기 생성.

## 🟡 Medium — 완성도 향상

### M1. BGM 부재 + 무음 구간 과다
**증상**:
- 오디오 트랙이 내레이션 단독 (mean_volume −26.1 dB, max −5.9 dB). 정적인 톤.
- `silencedetect=noise=-40dB:d=0.3` 결과 **1초 이상 정적 구간이 14개**, 누적 약 18~20초 → 전체 89초의 약 20%가 침묵.
- ElevenLabs 출력 사이 또는 컷 사이 silence padding이 원인으로 추정.

**개선**:
- 무음 구간 자동 트리밍 — 0.3초 이상 silence는 0.25초로 압축 (`ffmpeg silenceremove` 또는 후처리 파이프라인).
- 잔잔한 lo-fi/뉴스 인트로 BGM을 −24 ~ −20 dB로 깔고, 내레이션과 사이드체인 덕킹(−6dB) 적용.
- 사운드 이펙트(SFX): 인트로 woosh, "사상 최고치" 등장 시 sparkle SFX.
- 라우드니스 정규화 −16 LUFS (`ffmpeg loudnorm`).

### M2. 장면 전환이 hard cut 일색
**증상**: 19개 segment 모두 컷 전환. 페이드·매치컷·리듬 변화 없음.

**개선**: 호흡(문장) 단위로 80~120ms cross-dissolve. 단, hook 구간은 hard cut 유지.

### M3. 자막 가독성·컨트라스트
**증상**: 회색(#~C0) 배경 위 흰 자막 + 검정 외곽선이 일부 컷에서 명도 대비 4.5:1 미만으로 보임. 컬러 일러스트(frame_010, frame_018) 위에서 특히 약함.

**개선**:
- 자막 뒤에 반투명 검정 박스(`rgba(0,0,0,0.55)` + blur 8px) 적용.
- 폰트 굵기 700 이상, 외곽선 6px 이상.
- 폰트 패밀리 Pretendard 또는 Toss Product Sans Bold로 통일.

### M4. 아웃트로 CTA가 부실
**증상**: frame_030에서 "더 자세한 분석을 원하시면 말씀해 주세요"로 끝나지만, 화면 중앙은 여전히 데이브리핑 배너가 가리고 있고 CTA에 어울리는 버튼/링크/구독 유도 비주얼이 없음.

**개선**:
- 마지막 1.5초는 풀스크린 아웃트로 카드: "구독 + 알림 설정 / 내일 8시 다시 만나요" + 채널 핸들.
- `show_outro=true`로 토글하고 outro 컴포넌트 별도 빌드.

### M5. "MD2Short" 워터마크가 디자인 톤과 분리됨
**증상**: 산세리프 흰 글자가 검은 띠 위에 단독 노출 → "샘플 영상" 같은 인상. 도구 이름은 시청자에게 무의미.

**개선**:
- 도구명("MD2Short") 대신 채널 브랜딩(`큰돌의 터전`)으로 교체.
- SVG 마크 + 워드마크로 다듬고 우상단 코너로 이동, 투명도 60%.

## 🟢 Low — 마감 디테일

### L1. 컬러 시스템 부재
- 일러스트마다 배경 회색 톤(#B5B6B5 ~ #C8C8C7)이 미세하게 다름 → 컷마다 깜빡임처럼 보임.
- 브랜드 컬러 1차/2차/엑센트 3색을 `docs/architecture.md`에 정의하고 모든 일러스트·자막·배너에 적용.
  - 예: Primary `#3182f6` (Toss blue) / Accent `#FFD400` (warning yellow) / BG `#F5F5F5`.

### L2. QC 리포트가 "비주얼 품질"을 검사하지 않음
- `qc_report.json`은 sync/duration 두 가지만 본다(둘 다 pass).
- 추가 체크 제안:
  - `title_overlay_max_duration_ms` ≤ 2000
  - `caption_max_chars_per_line` ≤ 14
  - `caption_line_count` ≤ 2
  - `unique_asset_ratio` ≥ 0.7 (동일 일러스트 반복 방지)
  - `silence_ratio` ≤ 0.10 (무음 누적 비율)
  - `total_duration_ms` ≤ 60000 (숏폼 길이 가드)

### L3. 마지막 0.4초 잉여 프레임
- audio_duration = 89,722ms / final_mp4 = 89,771ms / last_word_end = 89,339ms.
- 마지막 단어 종료 후 약 400ms는 결국 정적 + 무음. 아웃트로 카드로 활용하거나 trim.

### L4. 파일 크기 최적화
- 30MB / 90초 = 약 2.4 Mbps. 숏폼 업로드용은 1.8~2.2 Mbps로 낮춰도 무손실 체감.
- `-crf 23 -preset slow` 또는 two-pass 인코딩 옵션을 후처리에 추가.

## 🛠 구현 메모

- **타이틀 오버레이 수정**은 `props.json` 생성 단계(`src/md2short/edit/props_builder.py` 또는 동급 파일)에서 `title_overlay.end_ms` 필드를 조정.
- **자막 청크 분할 로직**은 `chunk_size=14`(status.json:14) 상수 대신 `wrap_korean_caption(words, max_chars=14, max_lines=2)` 함수로 교체. 어절 경계 우선, 그 다음 문장부호, 마지막에 글자수.
- **숫자·영문 원본 표기**는 Script Engine 출력에 `display_text`(자막용) / `spoken_text`(TTS용) 두 필드를 분리하고, 자막 렌더 시 `display_text`를 사용. TTS는 `spoken_text`로 호출.
  - ElevenLabs 호출 직전 `ko_number_to_hangul()` 같은 유틸로 숫자만 한글화.
- **무음 트리밍**은 ElevenLabs 응답 직후 단계에 `ffmpeg silenceremove=stop_periods=-1:stop_duration=0.25:stop_threshold=-40dB` 패스 추가.
- **QC 비주얼 체크**는 `qc_report.json` 스키마에 `visual_checks[]` 배열을 추가하고 `qc_passed` 게이트에 포함. 실패 시 자동 재빌드 또는 알림.
- **metric_card 컴포넌트**는 Remotion `<Sequence>` 내부에 `<MetricCard ticker="S&P 500" value="5,723.0" change="+0.93%" trend="up" />` 형태로 분리. 마크다운 파서가 `숫자+단위` 토큰을 발견하면 자동 인서트.
