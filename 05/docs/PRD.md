# 할일 + 계획 관리 앱 PRD
## 1. Why — 목적

단순 할일 앱은 "해야 할 일"만 관리하고, "왜 하는지"와 "어디로 가는지"를 연결하지 못합니다.
이 제품은 단기 실행(할일)과 중기/장기 목표(주간, 1년, 3년)를 하나의 구조로 연결하여
사용자가 방향성과 실행을 동시에 관리할 수 있도록 합니다.

---

## 2. Who — 타겟 사용자
- 자기계발을 하는 개인 사용자 (대학생, 취준생, 주니어 개발자) 
---

## 3. User Flow

### 최초 사용 
1. 1년 목표 기반으로 이번 주 계획 작성
2. 주간 계획 기반으로 할 일 생성

### 일상 사용

1. 대시보드에서 이번 주 계획 확인
2. 할 일 생성 / 수정
3. 드래그앤드랍으로 상태 변경 (todo → doing → done)
4. 완료 작업이 주간 진행률에 자동 반영

---

## 4. 기능 요구사항

### P0 — 핵심 기능

**할 일 관리**
- 할 일 생성 / 수정 / 삭제
- 상태 변경 (todo / doing / done)
- 드래그앤드랍으로 상태 이동

**주간 계획**
- 이번 주 목표 생성 (최대 3~5개)
- 요일별 할 일 배치
- 주간 메모 작성

**목표 관리**
- 1년 목표
- 목표별 설명 작성

**연결 구조**
- 할 일 → 주간 계획 → 1년 목표

---

### P1 — 추가 기능

- 우선순위 설정 (High / Medium / Low)
- 마감일 설정
- 주간 진행률 자동 계산 (%)
- 목표별 진행률 표시
- 주간 회고 작성
